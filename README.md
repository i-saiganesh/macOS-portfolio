# macOS Portfolio.

A macOS style portfolio website with draggable windows and interactive UI, built using React and Vite.

Live Website: https://saiganeshs.vercel.app/
Repository: https://github.com/i-saiganesh/macOS-portfolio

## Features
- macOS style desktop interface
- Draggable application windows
- Project and resume sections
- Smooth animations
- Responsive design

## Tech Used
- React
- Vite
- CSS

## Installation
git clone https://github.com/i-saiganesh/macOS-portfolio.git
cd macOS-portfolio
npm install
npm run dev

## Build
npm run build
